== Description ==

This plugin allows you to require users to log-in to see content, and also specify which pages and which roles are allowed to see that pages. You can contact me in <a href="http://www.codebox.es">codebox.es</a>

== Installation ==


1. Upload wp-private-pages folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Choose your private pages and roles!
